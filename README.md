## 📱 Highlight Project: Balanceè USSD Service

A **Spring Boot USSD application** integrated with **Africa's Talking API**.

### 🚀 Overview
- Service Code: *347*426#
- Built with Spring Boot + Java 21
- Real-time USSD interaction system
- Handles authentication, wallet, and transactions

### 🎯 Features
- 📲 Interactive USSD menu system  
- 💰 Wallet balance & transactions  
- 🔐 Session management  
- ⚡ Fast response time (~0.8s)

### ⚠️ Current Status
- ✅ Fully working locally  
- ✅ API endpoints tested  
- ❌ Production connectivity issue (Africa's Talking timeout)

### 🔧 Key Endpoints
- `/ussd` – Main callback  
- `/ussd/health` – Health check  
- `/ussd/test` – Testing  
- `/ussd/simulate` – Simulation  

### 🧠 Tech Used
- Spring Boot  
- Africa's Talking API  
- Redis (session handling)  
- Ngrok
